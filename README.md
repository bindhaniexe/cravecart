# CraveCart – Food Ordering Platform

CraveCart is a full-stack food ordering platform designed to connect customers with restaurants through a seamless and user-friendly digital experience. The platform enables users to browse restaurants, explore menus, place orders, manage carts, and track their ordering experience efficiently.

Built using Spring Boot and React, CraveCart follows modern software development practices with a scalable architecture, secure authentication, and responsive user interfaces.

---

## 🚀 Features

### 👤 User Features

* User Registration and Login
* Secure Authentication and Authorization
* Browse Restaurants and Menus
* Search and Filter Food Items
* Add Items to Cart
* Place and Manage Orders
* View Order History
* Personalized Food Recommendations

### 🏪 Restaurant Features

* Restaurant Listing Management
* Menu Management
* Order Processing
* Inventory Updates

### 🛠️ Admin Features

* User Management
* Restaurant Management
* Order Monitoring
* Analytics Dashboard

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/cravecart.git
cd cravecart
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
